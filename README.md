# Agent Skills

Skills collection for agent workflows.

The repository and project name are `agent-skills`, and the local skill
identifiers in this repo are aligned to current published names.

The runtime packages used in this repository are `genai-calling` and
`@gravtice/agent-swapcloud`.

## Repository Structure

- `skills/find-skills/SKILL.md`: Skill discovery helper
- `skills/genai-calling/SKILL.md`: `genai-calling` skill for the `genai-calling` runtime
- `skills/agent-swapcloud/SKILL.md`: `agent-swapcloud` skill for the `@gravtice/agent-swapcloud` runtime
